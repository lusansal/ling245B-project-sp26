Rocio Sanchez Salcido

Stanford University, Spring 2026

## Overview

This project is a replication of Experiment 5 from Costa & Santesteban (2004), investigating lexical access and language switching in bilingual speakers. The original study used verbal picture-naming responses. This replication adapts the paradigm for online data collection by replacing verbal responses with keyboard responses.

## Task design

On each trial, participants are cued to respond in either English or Spanish via a coloured dot:

- Red dot → respond in English
- Blue dot → respond in Spanish

A picture is then displayed and participants press the key corresponding to the first letter of the picture's name in the cued language. For example, a picture of a dog requires pressing **D** in English and **P** in Spanish (perro).

### Stimuli

Ten pictures were used:

| Image | English | Spanish |
|-------|---------|---------|
| Apple | A | M (manzana) |
| Carrot | C | Z (zanahoria) |
| Dog | D | P (perro) |
| Rain | R | L (lluvia) |
| Window | W | V (ventana) |
| Table | T | M (mesa) |
| Cheese | C | Q (queso) |
| Hat | H | S (sombrero) |
| Knife | K | C (cuchillo) |
| Leaf | L | H (hoja) |
